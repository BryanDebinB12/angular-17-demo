Control flow : For

The trackBy function is used to improve performance improve performance of ngFor. It is mandatory in the new syntax. In doubt, use 
track @index
.

Old syntax

{{ item.label }}

New syntax. Using $index

@for (item of list; track $index) { {{ item.label }} }

New syntax. Using trackBy properly

@for (item of list; track item.id) { {{ item.label }} }
