Defer

Only standalone components, directives, and pipes can be deferred.

Use cases
  • Decrease initial bundle size, e.g. when component has a huge code base
  • Component that make a lot of computations/calls when instantiated
One can define events that trigger the loading of the component, for example on hover, click, timer, conditions, viewport (when user first sees the component). See documentation
